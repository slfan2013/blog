Setup an R-calculating Website

Sign out

You may find this blog useful if you know how to use R, and would like to provide some statistical analysis tool-website for people to use.

Step by step, this blog shows how to set up such a website/blog that is capable of

  • performing calculation using statistical software, R.
  • easily hosted through a free server, which is opencpu cloud.
  • 100% control over the web-design.

Basically, you could host a blog/website having a similar concept with Shiny but with much more flexibility and calculation power. Actually, this entire blog is done this way.

1. Create an R package

First you have to create an R package that contain all the functions that you'll need in your website. Consider R as a back-end language (like pt). Now, let's assume that we would like to create a website that could do simple linear regression.

  1. Open RStudio, click "File" -> "New Project"
  2. Click "New Directory" --> "R package"
  3. Set package name as "simple_linear_regression_web"
  4. Remember to brower your R package to where you would like to put it. (For example, ~/Github)
  5. Write a R simple linear function. You can download my functino script here. Don't worry about the parameters, I'll explain soon.
  6. Click "Build". Congraduations! You have just set up your back-end calculation environment! Let's keep going!

2. Write a website

Here you can design your website whatever you like.

You have to have some website development skill. Actually, this is pretty simple to pick up, you just have to know html and javascript. I learned most of the website development skill (very basic but enough) from W3School.

Also, you could pick up a web theme by Googleling it. For simplicity, let's download my template for now.

  1. Open your R package using file explorer. (For example, ~/Github/simple_linear_regression_web).
  2. put what you just downloaded in this folder. Note, the inst must be in the root folder of your package.
  3. Congraduations! You have just set up your front-end UI successfully! Now, let's connect back-end to front-end.

3. Set up OpenCPU

OpenCPU is the magic that connects your back-end calculation environment (R) to your front-end UI (html, javascript, etc)! And setting it up is super easy. You just have to a few steps in both back-end and front-end.
